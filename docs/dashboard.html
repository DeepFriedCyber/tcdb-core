Quick Start

Get started with the TopoPersist API in minutes

1. Install the SDK

pip install tcdb-client

2. Initialize the Client

from tcdb import TCDBClient

client = TCDBClient(api_key="your-api-key")

3. Make Your First Request

result = client.topological_signature(
    data=your_data,
    method="vietoris_rips"
)
print(result)
